inline

Initializes a new instance of the CoreTweet.Streaming.StreamingParameters class.

Avaliable parameters:

*Note: In filter stream, at least one predicate parameter (follow, locations, or track) must be specified.

bool stall_warnings (optional) : Specifies whether stall warnings should be delivered.

string, IEnumerable<string> follow (optional*, required in site stream, ignored in user stream) : A list of user IDs or comma separated string of ones, indicating the users to return statuses for in the stream.

string, IEnumerable<string> track (optional*) : Keywords to track. Phrases of keywords are specified by a list or comma separated string.

string, IEnumerable<string> location (optional*) : A list of longitude,latitude pairs or comma separated string of ones, specifying a set of bounding boxes to filter Tweets by. example: "-74,40,-73,41"

string with (optional) : Specifies whether to return information for just the authenticating user (with => "user"), or include messages from accounts the user follows (with => "followings").
