Provides a set of methods for the wrapper of GET/POST mutes/users.

UserResponse CoreTweet.Rest.MutesUsers.Create ( params Expression< Func< string, object >>[]  parameters)
inline

Mutes the user specified in the ID parameter for the authenticating user.

Returns the muted user in the requested format when successful.

Returns a string describing the failure condition when unsuccessful.

Actions taken in this method are asynchronous and changes will be eventually consistent.

Note: Providing either screen_name or user_id is required.

Available parameters:

- string screen_name (optional)

- long user_id (optional)
