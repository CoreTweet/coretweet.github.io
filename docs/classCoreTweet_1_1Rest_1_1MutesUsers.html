User CoreTweet.Rest.MutesUsers.Create ( params Expression< Func< string, object >>[]  parameters)
inline

Mutes the user specified in the ID parameter for the authenticating user.

Returns the muted user in the requested format when successful. Returns a string describing the failure condition when unsuccessful.

Actions taken in this method are asynchronous and changes will be eventually consistent.

Note: Providing either screen_name or user_id is required.

Avaliable parameters:

string screen_name (optional) : The screen name of the potentially muted user. Helpful for disambiguating when a valid screen name is also a user ID.

long user_id (optional) : The ID of the potentially muted user. Helpful for disambiguating when a valid user ID is also a valid screen name.

Returns
The user.
