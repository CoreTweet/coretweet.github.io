Status CoreTweet.Rest.Favorites.Destroy ( params Expression< Func< string, object >>[]  parameters)
inline

Un-favorites the status specified in the ID parameter as the authenticating user. Returns the un-favorited status in the requested format when successful.

This process invoked by this method is asynchronous. The immediately returned status may not indicate the resultant favorited status of the tweet. A 200 OK response from this method will indicate whether the intended action was successful or not.

Avaliable parameters:

long id (required) : The numerical ID of the desired status.

bool include_entities (ooptional) : The entities node will be omitted when set to false.

IEnumerable<Status> CoreTweet.Rest.Favorites.List ( params Expression< Func< string, object >>[]  parameters)
inline

Returns the 20 most recent Tweets favorited by the authenticating or specified user.

Avaliable parameters:

long id (optional) : The ID of the user for whom to return results for.

string screen_name (optonal) : The screen name of the user for whom to return results for.

int count (optional) : Specifies the number of records to retrieve. Must be less than or equal to 200. Defaults to 20.

int since_id (optional) : Returns results with an ID greater than (that is, more recent than) the specified ID. There are limits to the number of Tweets which can be accessed through the API. If the limit of Tweets has occured since the since_id, the since_id will be forced to the oldest ID available.

int max_id (optional) : Returns results with an ID less than (that is, older than) or equal to the specified ID.

bool include_entities (optional) : The entities node will be omitted when set to false.

Returns
The statuses.
